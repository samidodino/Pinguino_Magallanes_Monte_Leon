LatinR 2025. Modelado del éxito reproductivo del pingüino de Magallanes mediante R e imágenes satelitales: un enfoque multinomial desde el Parque Nacional Monte León, Argentina.

Este repositorio contiene el flujo de trabajo desarrollado que integra datos de campo y variables satelitales para analizar el éxito reproductivo del pingüino de Magallanes en el Parque Nacional Monte León, Argentina. Este proyecto corresponde una serie temporal de 15 años de datos.

Archivos:
1) data_ambiental_PM_Monte_Leon: contiene el flujo de trabajo para extraer datos ambientales durante el cuidado temprano de pichones usando 15 años de datos.
2) PM_exito+ambiental: contiene el modelo multinomial utilizado para analizar la relación entre el éxito reproductivo y las condiciones ambientales.
